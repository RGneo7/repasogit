Curso Git

git init - iniciamos el repositorio

git status -s /comando para identificar el estado de los archivos respecto a git
-si aparecen archivos con doble signo de interrogacion inplica que no estan bajo seguimiento
A - ya se le esta haciendo seguimiento
M - a sido modificado

git add nameArchivo / de esta forma instanciamos un archivo en particular
git add . /para que agregue todos los archivos

git commit -m "comienzo del proyecto" /instanciamos el proyecto - la m es el mensaje que queremos guardar con la instantanea

git log --oneline - .e muestra los commit guardados

git reset --hard IdCommit /nos permite retornar a un commit en particular

tanto git add como git commit pueden ejecutarse con un mismo comando
git commit -am "descripcion"

En caso de querer cambiar la descripcion
git commit --amend

para subir los archivos al repositorio usamos
git remote add github https://github.com/RGneo7/repasogit.git git remote set-url origin git@github.com:ppreyer/first_app.git

para actualizar los cambios guardados en github en local
git pull

Etiquetas
git tag nameTag -m "descripcion"
git push --tags

Ramas

crear rama
git branch nameRama

para listar las ramas
git branch

para movernos entre ramas
git checkout nameRama
