Curso Git

git init - iniciamos el repositorio

git status -s /comando para identificar el estado de los archivos respecto a git
-si aparecen archivos con doble signo de interrogacion inplica que no estan bajo seguimiento
A - ya se le esta haciendo seguimiento
M - a sido modificado

git add nameArchivo - de esta forma instanciamos un archivo en particular

git commit -m "comienzo del proyecto" /instanciamos el proyecto - la m es el mensaje que queremos guardar con la instantanea
